# 介绍

## TypedAnt是什么?
`TypedAnt`是一门强类型的编译型语言, 主要使用rust开发.<br>
旨在`可用，好用`.

## TypedAnt的设计?
`TypeAnt`的设计优美, 语法明晰.<br>
适合`learn as the first pl`, 也适合当作补充和娱乐,<br>
可用于高性能要求和低条件环境, (in a word,a `great` pl).

## TypedAnt所受的称赞?
知名的LKBaka曾这么称赞TypedAnt:
> `ant_cranelift_compiler` 是一个可用，好用的可执行文件编译器
